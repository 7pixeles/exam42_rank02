# Exam Rank Practice - Campus 42

This repository contains my practice exercises for the **Exam Rank** at **Campus 42**. The main goal is to improve my programming skills and solve typical problems that appear in the school's exams.

---

## Contents

The repository is organized as follows:

* `ex00/` - Level 00: Foundation exercises
* `ex01/` - Level 01: Intermediate Challenges
* `ex02/` - Level 02: Advanced Problems
* `ex03/` - Level 03: LEVEL BOSS - Expert Level
* ...
  *(Each folder contains the corresponding code and an internal comment explaining the logic.)*

## How to use

1. Clone the repository:

   ```bash
   git clone https://github.com/7pixeles/exam42_rank02.git
   ```
2. Enter the folder of the exercise you want to review:

   ```bash
   cd ex00
   ```
3. Check the code and comments to understand the solution for each exercise.
